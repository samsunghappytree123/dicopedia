# wikifordiscord
제2회 한디리 해커톤 출품예정작

## 정보
1. 팀명 : ``뱀이불과 함께 꿀잠을 꿈꾼다``
2. 팀원 : ``Happytree Samsung#7612``, ``뱀이불#6109``
3. 주제 : ``디코백과 - 디스코드의 다양한 정보를 하나의 백과사전 봇으로``

## 사용 방법 - 준비중
1. ``Python 3.8.5 (또는 그 이상)``을 설치 후, **사용하는 모듈을 설치**(``pip install -r requirements.txt``)합니다.
2. ``main.py`` 파일을 실행합니다.
